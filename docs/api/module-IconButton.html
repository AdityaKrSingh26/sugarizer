Module: IconButton

This is an button component to display button with label and icon

Props

Name Type Default value Required? Description
text String 'empty String' No text label of button
id Number - Yes Id of the icon component of button
svgfile String - Yes Url of svg file of icon in button
color Number 512 No color index value of icon in button
size Number 55 No size in `px` of icon in button
x Number 0 No left-right margin of icon in button
y Number 0 No top-bottom margin of icon in button
disabled Boolean false No true for disabling the button

Data

Name Type Default value Description
iconData Number - to change the icon file url data
textData String 'empty String' to change the label data
disabledData Boolean false to change the diability condition

Events

Name Payload Type Description
buttonClick Object Emit buttonClick event to perform required action
