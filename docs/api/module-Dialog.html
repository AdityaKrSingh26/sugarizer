Module: Dialog

This is an modal component for different settings overlay pages

Props

Name Type Default value Required? Description
searchField Boolean null No to show search-bar in the toolbar
okButton Boolean null No to show ok button in the toolbar
cancelButton Boolean null No to show cancel button in the toolbar
iconData String null No stores the url of svg file of icon in the toolbar
titleData String null No stores the title text of feature page in the toolbar

Data

Name Type Default value Description
showDialog Boolean false condition to show or hide the dialog modal

Events

Name Payload Type Description
onCancel emit cancel event to close the dialogBox
onOk emit ok event to save the changes and close the dialogBox
searchInput String emit inputString of the search-bar whenever its changes
